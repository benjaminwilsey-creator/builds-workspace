# ReelForge

Automated BookTok affiliate content pipeline. Discovers trending romance/romantasy books,
generates 30-second video reels, and delivers download-ready videos with Amazon affiliate
captions for human scheduling in Meta Business Suite.

**Design phase complete. Phase 0 in progress.**
