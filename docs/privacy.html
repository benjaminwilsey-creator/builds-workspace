Privacy Policy

ReelForge — Last updated: March 2026

1. Overview

ReelForge is an internal tool used to manage and post BookTok video content to a single TikTok creator account. This policy describes how data is handled in connection with the TikTok API integration.

2. What Data We Access

ReelForge requests the video.upload scope from TikTok, which allows the tool to upload video drafts to the administrator's TikTok account. No other TikTok account data is accessed or read.

3. How Data Is Stored

OAuth access tokens and refresh tokens obtained from TikTok are stored in a secured cloud secrets manager (Google Cloud Secret Manager). They are never logged, displayed publicly, or shared with third parties.

4. Data Sharing

ReelForge does not sell, share, or transmit any user data to third parties. Data flows only between ReelForge's backend infrastructure and the TikTok API.

5. Data Retention

OAuth tokens are retained only as long as the TikTok integration is active. Tokens can be revoked at any time by disconnecting the app in TikTok's account settings.

6. Your Rights

As this tool is operated for a single administrator account, the administrator retains full control over all data. To revoke access, visit TikTok Settings → Security → Apps and Websites and remove ReelForge.
